# notas-atletas
Projeto teste git de Devstart-Senai

## Descrição

Aplicação em JavaScript que calcula a média válida descartando a maior e a menor nota e exibe o resultado no console.

- Como funciona

* Cada atleta possui 5 notas
* A maior e a menor nota são descartadas
* A média é calculada com as 3 notas restantes

- Como executar

! Opção 1 – Terminal (Node.js)

1. Instale o Node.js
2. No terminal, acesse a pasta do projeto
3. Execute:

```
node notas-atletas.js
```

4. O resultado aparecerá no terminal via `console.log`



## Testes
